# 📱 VEXA Show do bilhão

> An app to make your life easier

## Prerequisites
  • Flutter 3.3.4

## 🚀 Usage

Make sure you have [flutter](https://docs.flutter.dev/get-started/install) installed (`3.3.4`)

Just run the following command at the root of your project and answer questions:

```sh
flutter upgrade
```

```sh
flutter pub get
```

```sh
flutter run
```
